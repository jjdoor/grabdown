# Changelog

## 0.1.0 — 2026-06-12

Initial release.

- Copy a page **selection**, a **link**, or the **whole page** as clean Markdown.
- Entry points: right-click context menu, toolbar popup, and a keyboard command.
- Hand-written, zero-dependency HTML→Markdown converter: headings, emphasis,
  links, images, inline + fenced code (with language detection), nested lists,
  blockquotes, rules, strikethrough, highlight, and GFM tables.
- Zero data collection, no host permissions, no account.
- One MV3 codebase for Chrome, Edge, and Firefox.
